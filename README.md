## 송민정, 백창룡. (2025). 그래프 기반 신경망을 이용한 고차원 시계열 자료의 군집 탐지. 한국데이터정보과학회지, 36(5), 799-811. 10.7465/jkdi.2025.36.5.799

### Abstract
고차원 시계열 자료의 군집 분석에서 Guðmundsson과 Brownlees (2021)의 스펙트럴 클러스터링 기반 VAR-Blockbuster 알고리즘은 이론적 정당성과 계산 효율성으로 인해 널리 사용되고 있다. 그러나 해당 방법은 SB-VAR (Stochastic block-vector autoregression) 모형에 의존하므로 적용 범위에 제약이 존재한다. 본 연구는 이러한 제한을 보완하고자 모형 가정에 덜 민감한 딥러닝 기반 그래프 신경망인 Adaptive graph encoder(AGE)를 활용하여 고차원 시계열 군집 탐지에 대한 대안을 제시한다. 모의실험을 통해 표본 크기가 충분한 경우에는 VAR-Blockbuster가 안정적인 성능을 보였으나 표본수가 제한적인 환경에서는 AGE가 상대적으로 우수한 군집 탐지 성능을나타냈다. 또한, AGE기반 방법을 S&P 500 지수 내 금융 및 소비재 섹터 종목 그리고 서울시 자치구별 주택 매매가격지수에 적용한 결과 구조적으로 해석 가능한 군집을 도출할 수 있음을 실증적으로 확인하였다.
